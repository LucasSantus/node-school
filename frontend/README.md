<h3>:computer: Rodando o Frontend
  
> **Observação:** Para que o projeto funcione corretamente, siga todos os passos disponibilizados abaixo.

<h4>Instalando Dependencias</h4>

```
npm install
```

**Construindo o Projeto**

```
docker-compose up --build
```

**Rodando o Projeto**

```
docker-compose up
```

**Acessando o Projeto**

http://localhost:3000/
